# Program Window

A Now Playing overlay styled to look like a classic desktop application window — complete with a title bar, window frame chrome, and system-style typography. The nostalgic UI dressing wraps around the track info and album artwork, giving the whole widget the feel of a real media player running on a vintage operating system.

A natural fit for retro PC gaming streams and tech-history content, but versatile enough for any layout that already leans into pixelated, skeuomorphic, or old-school computer aesthetics.

This overlay ships as part of the default Angels-NowPlaying bundle and is available to all users without any additional installation.

## Scene Recommendations

- Retro PC gaming or emulation streams
- Tech history or computing-focused content
- Windows 95 / early 2000s nostalgic stream aesthetics
- Any scene pairing well with system fonts and classic UI elements
