# Glassmorphism 2

A refined second variant of the frosted glass Now Playing overlay. It shares the same transparent, blurred-glass visual language as Glassmorphism 1 but features adjusted layout proportions, spacing, and styling for a slightly different composition and feel.

Both Glassmorphism variants use the same core aesthetic — if one feels like it's slightly too large or too small for your scene, try the other. They are interchangeable in most layouts and differ mainly in scale and internal arrangement rather than style.

This overlay ships as part of the default Angels-NowPlaying bundle and is available to all users without any additional installation.

## Scene Recommendations

- When Glassmorphism 1 doesn't quite fit your scene proportions
- Stream layouts with dark or semi-transparent scene backgrounds
- Modern, minimal scenes where a frosted glass widget feels at home
- Scenes where a progress bar is not needed
