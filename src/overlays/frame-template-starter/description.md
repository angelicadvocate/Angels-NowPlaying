# Template Starter

A fully working, heavily commented developer overlay for building your own Angels-NowPlaying overlays. Copy this folder as the base for any new overlay — every file is annotated to explain what it does, what hooks are available, and what the app and `common.js` expect to find.

This is not intended as a production overlay for viewers. It is a learning tool and starting point for overlay authors.

## Who this is for

- Developers building a custom overlay to submit to the store
- Anyone contributing a new bundled overlay to Angels-NowPlaying
- Anyone curious about how the overlay system works end-to-end
