# Horizontal Wide

A bold, wide horizontal Now Playing overlay with larger album artwork and a longer, more prominent playback progress bar. Where Horizontal Classic is understated and tucked away, this variant makes a stronger visual statement while keeping the same familiar horizontal layout.

Best suited for layouts where the current track deserves dedicated screen space — music-focused streams, DJ sets, or scenes where the Now Playing widget is a focal point rather than background information.

This overlay ships as part of the default Angels-NowPlaying bundle and is available to all users without any additional installation.

## Scene Recommendations

- Music-focused streams or live DJ sets
- Dedicated "Now Playing" scene transitions
- Full-width lower-third or banner placements
- Streams where the current track is a central part of the content
