About

Angel's-NowPlaying is a customizable, browser-based overlay for streamers. Perfect for OBS, it displays your current song, album art, and artist information in beautiful, animated layouts.

Choose from multiple theme designs including retro cassette tapes, spinning vinyl records, glassmorphism, and neon cyberpunk aesthetics. Each template is fully customizable to match your stream's visual style.

Created by: AngelicAdvocate

Made with ❤️ for content creators

Dependencies: Tuna, OBS, VLC

Configure HTTP Server

Optionally serve your overlays over HTTP instead of file://. Useful for store / third-party overlays that need fetch() or for running OBS on a different machine.

Port the local server will bind to. Range 1024–65535. Default is 8253.

When OFF, the server is loopback-only (127.0.0.1). When ON, it binds to 0.0.0.0 so other devices on your LAN can reach it.

This opens your overlays to any device on your local network. Only enable this if you run OBS on a different machine. Your operating system may show a firewall prompt.
